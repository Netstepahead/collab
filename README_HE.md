## 🚀 התחלה מהירה

### דרישות מקדימות
- Node.js 18+ ו-npm
- חשבון Supabase (חינם!)

### 1. התקנה

```bash
cd web
npm install
```

### 2. הגדרת Supabase

#### צור פרויקט Supabase:
1. עבור ל-[supabase.com](https://supabase.com)
2. לחץ "New Project"
3. מלא פרטים
4. חכה ~2 דקות

#### הרץ את ה-Schema:
1. ב-Supabase, עבור ל-**SQL Editor**
2. העתק את התוכן מ-`supabase/schema.sql`
3. הדבק והרץ
4. זה יוצר את כל הטבלאות!

#### קבל את ה-Credentials:
1. עבור ל-**Settings** → **API**
2. העתק:
   - Project URL
   - anon/public key

#### צור `.env.local`:

```bash
cp .env.local.example .env.local

# ערוך .env.local והוסף:
NEXT_PUBLIC_SUPABASE_URL=https://your-project.supabase.co
NEXT_PUBLIC_SUPABASE_ANON_KEY=your-anon-key-here
```

### 3. הרץ את השרת

```bash
npm run dev
```

פתח [http://localhost:3000](http://localhost:3000)
